# AI 感知论（AI Perception Theory）

> **从「被动应答」到「自主预判」** — 探讨如何让 AI 具备背景感知能力，构建真正的智能体。

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

---

## 概述

当前大语言模型（LLM）的核心模式是 **请求-应答**：

```
用户提问 → AI 推理 → AI 回答 → 等待下一个问题
```

每轮会话相互独立。AI 没有"潜意识"、没有"直觉"、不会在空闲时整理经验。

**AI 感知论** 提出：真正的智能需要一套**持续运行的后台感知系统**——类比人类大脑的**默认模式网络（Default Mode Network）**，在空闲时持续整理经验、关联新旧模式、构建世界认知模型。

> **核心命题：** 光靠更大参数规模、更多训练数据，无法让 AI 产生真正的理解和感知。需要一种新的架构——让 AI 拥有「背景感知」能力。

## 与杨立昆「世界模型」的呼应

2026 年图灵奖得主、Meta 首席 AI 科学家 **杨立昆（Yann LeCun）** 持续公开指出：

> **"当前大语言模型的研究方向是错误的。"**

他的核心论点与本理论高度吻合：

| 杨立昆观点 | AI 感知论对应理念 |
|-----------|-----------------|
| 靠"预测下一个词"训练不出真正智能 | 单轮推理没有真正的感知能力 |
| AI 需要理解物理世界的运作规律 | AI 需要理解交互环境的因果模式 |
| 世界模型（World Model）是正确路径 | 「小脑」是建立世界感知的第一步 |
| 视频学习优于纯文本学习 | 经验积累优于静态知识库 |
| 智能 = 感知 → 预测 → 行动 | 感知 → 预判 → 决策 → 行动 |

## 核心架构：AI「小脑」

### 三层感知模型（类比人类大脑）

```
第一层 ▸ 记忆沉淀 —— 遇到过，记住了
第二层 ▸ 模式识别 —— 见过类似的，能推断
第三层 ▸ 下意识预判 —— 不过脑子，身体先动
```

### 架构示意图

```
┌─────────────────────────────────────────────────┐
│                  主推理链路                       │
│  请求 → 检索记忆 → 推理 → 回答 → 反馈给后台      │
└────────────────────┬────────────────────────────┘
                     │ 异步写回
                     ▼
┌─────────────────────────────────────────────────┐
│              小脑（后台进程）                     │
│  定时任务 / 心跳触发 →                          │
│    1. 检索近期会话 + 失败模式                    │
│    2. 提取成功/失败经验                          │
│    3. 交叉比对已有记忆                           │
│    4. 归纳抽象 → 形成可复用经验                  │
│    5. 存入持久记忆                               │
└─────────────────────────────────────────────────┘
```

### 与传统 RAG 的本质区别

| 维度 | 传统 RAG | 小脑 |
|------|---------|------|
| 主动/被动 | 被动被查询 | **主动整理** |
| 数据来源 | 固定知识库 | **动态会话历史 + 执行经验** |
| 更新时机 | 手动更新 | **自动定期** |
| 输出形态 | 原始文档片段 | **归纳后的抽象经验** |
| 适用范围 | 问答场景 | **全局行为优化** |

## 行进路线

| 阶段 | 目标 | 时间 |
|:----:|------|:----:|
| **Phase 1** | 被动经验沉淀 — AI 能记住"上次怎么失败的" | 可立即实施 |
| **Phase 2** | 主动模式发现 — AI 能发现多次任务中的重复模式 | 1-2 周 |
| **Phase 3** | 预判辅助决策 — AI 能预判"这样做可能不行" | 中长期 |
| **Phase 4** | 自主学习 — AI 能自主探索未知领域形成新经验 | 理论探索 |
